# IoT-Kel3-BlueJammer
# PROJECT IoT: BlueJammer esp32

## Deskripsi
elektronik yang bertujuan untuk membuat perangkat pengganggu sinyal Bluetooth berdaya rendah (low-power). Proyek ini umumnya menggunakan mikrokontroler ESP32 yang dikombinasikan dengan modul radio nRF24L01 untuk memutus koneksi antara dua perangkat Bluetooth, seperti ponsel dan headset.


## Hardware Preparation
1. Modul ESP32
2. Nrf24
3. TP4056
4. Switch
5. Kapasitor UV10 50V
6. Baterai Li-Po 3.7 V
7. Oled

## Library & Anything
1. GitHub

## Hasil
Alat ini memancarkan sinyal gangguan pada frekuensi Bluetooth yang menyebabkan perangkat target kehilangan koneksi
